<!--@installation()-->
## Installation

```sh
npm install --save mos-plugin-license
```
<!--/@-->

## Usage

Add this code snippet to the footer of your `README.md`

```md
<!--@license()-->
<!--/@-->
```

Run `mos` in the terminal.

You'll get a license section with the license and author specified in the `package.json`.

## API

`license()`
